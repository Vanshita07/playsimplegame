# playsimplegame

## This repository is for the PlaySimple Games UI Assessment.
### The repository contains a web interface and to exexute this project pull the repository and run supercell.html file and  run HayDay.html file.
#### List of modules used in this project:
#### HTML: It is used to create the structure of the web page.
#### CSS: It is used to style the web page. Libraries like materialize and bootstrap is used to modify the website. 
#### Bootstrap and Materialize are used to make the webpage more attractive and used to make UI more familiar to the user.
#### These css libraries is very easy to use and avoid hard coding . By using these libraries , we can easily style our web pages by providing the particular keywords or tags in the class only in html code (i.e., it prevents us to add more css in the code.).
